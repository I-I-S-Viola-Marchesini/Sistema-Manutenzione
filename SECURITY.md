# Security Policy

## Supported Versions / Versioni supportate

Sistema Manutenzione è in continuo sviluppo, l'ultima versione disponibile dovrebbe essere sempre l'unica usata.

## Reporting a Vulnerability / Segnalare le vulnerabilità

Le vulnerabilità possono essere segnalate contattando direttamente [la scuola](https://www.iisviolamarchesini.edu.it/j2/).
